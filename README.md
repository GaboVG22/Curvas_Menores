# HidroSed Curvas · Cuencas Menores a 100.000 km² v1.1

Aplicación verificada para generar DEM, curvas de nivel KMZ y eje principal de cauce.

## Main file path

```text
app.py
```

## Mejoras incluidas

- Curvas desde 1 m.
- Curvas por teselas.
- Eje de cauce ingresado o generado desde DEM.
- Polígono de cuenca opcional.
- KMZ combinado compatible con App Modo DIOS.
- Metadata JSON.
- QA con puntaje objetivo > 8,7 cuando el paquete está completo.
